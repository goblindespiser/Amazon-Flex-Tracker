# flex-track

A local desktop Amazon Flex bookkeeping and tax-estimate tool built with Python, Tkinter, and SQLite. The current downloadable build is the v0.1.0 portable ZIP.

## Current features
- Daily log entry
- Other expense tracking
- Summary view
- Tax estimate view
- CSV export
- Local backups

## Notes
This is a public pre-release build. It is not tax filing software. Double check estimates. For State estimates, the only supported state at this time is Ohio. 

## Download & Installation
1. Click into "Releases" (if you cannot find Releases here is a direct link to the .zip: https://github.com/goblindespiser/flex-tracker/releases/tag/v0.1.0)
2. Click into the latest version, presently v0.1.0
3. Click "FlexTrack.zip" to begin download to desired destination folder
4. Extract zip
5. Open folder, double click FlexTrack.exe to run
