# flex-tracker

A local desktop Amazon Flex bookkeeping and tax-estimate tool built with Python, Tkinter, and SQLite.

## Current features
- Daily log entry
- Other expense tracking
- Summary view
- Tax estimate view
- CSV export
- Local backups

## Notes
This is a private working prototype. It is not tax filing software. Double check estimates.

## Download & Installation
1. In the right hand menu under the "About" section, click "Releases"
2. Click into the latest version, presently v0.1.0
3. Click "FlexTrack.zip" to begin download to desired destination folder
4. Extract zip
5. Open folder, double click FlexTrack.exe to run
